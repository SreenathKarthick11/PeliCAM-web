PeliCAM

A PyQt-based desktop app for visualizing deep learning model explanations with CAM, LIME, and bounding boxes.

Features Docs Download About

🚀 Features

  • 🖼️ Load Images & Models – Import images and PyTorch models easily.
  • 🔥 Apply CAM Techniques – Grad-CAM, Layer-CAM, and more.
  • ✏️ Draw Bounding Boxes – Compare ground truth vs. CAM outputs.
  • 🎯 LIME Explanations – Positive/negative influential features.
  • 💾 Save & Compare Outputs – Side-by-side tabbed comparison.

📖 Documentation

You can download the full PeliCAM Manual (PDF).

FAQ

  • Supported models? ResNet50, VGG16, VGG19 + custom PyTorch.
  • Image formats? PNG, JPEG, JPG.
  • Why CAM/LIME output missing? Ensure model, layer, and “Generate” are selected.
  • Can I compare outputs? Yes, each visualization opens in a separate tab.
