PeliCAM

A PyQt-based desktop app for visualizing deep learning model explanations using CAM, LIME, and bounding boxes.

Features Docs Download About

Features

  • 🖼️ Load Images & Models – Import images and PyTorch models easily.
  • 🔥 Apply CAM Techniques – Visualize with Grad-CAM, Layer-CAM, etc.
  • ✏️ Draw Bounding Boxes – Annotate and compare ground truth vs. CAM outputs.
  • 🎯 LIME Explanations – Explore positive and negative influential features.
  • 💾 Save & Compare Outputs – Store results, tabbed viewer for side-by-side comparison.

Documentation

FAQ

  • What models does PeliCAM support? Built-in (ResNet50, VGG16, VGG19) + custom PyTorch models.
  • Which image formats are supported? PNG, JPEG, JPG.
  • Why is my CAM/LIME output not showing? Ensure you selected model, layer, and pressed Generate.
  • Can I compare outputs? Yes, each visualization opens in a separate tab.
